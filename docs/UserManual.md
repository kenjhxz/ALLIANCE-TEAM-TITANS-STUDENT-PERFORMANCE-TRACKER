# Student Performance Tracker — User Manual

## 1) Accessing the System
- Open the web app in your browser.
- Use the role-specific login links (Student / Teacher / Admin).

## 2) Student Guide
### Profile & Account
- Update your name fields and change password in **Profile / Account**.

### Enrollment
- Use **Enrollment** to request disciplines.
- After approval, select the schedule for enrolled subjects.

### Grades
- Use **Grades** to view current grades and history.
- Export grades as CSV or PDF.

### Notifications
- Use the bell icon to view grade updates and other notifications.

## 3) Teacher Guide
### Offerings
- Select an assigned offering from the left sidebar.

### Grade Entry
- Enter prelim/midterm/finals and optional remarks.
- Click **Post** to create a grade or **Update** to edit.
- Use **Delete** to remove an entry.

### Notifications
- Teachers receive grade update notifications for confirmation.

## 4) Admin Guide
### School Setup
- Manage colleges, programs, disciplines, and semester load.

### Subject Offerings
- Create offerings and assign teachers.

### Enrollment Queue
- Approve/reject enrollment requests.

### User Management
- Search, edit, deactivate, or delete users.
- Add new teachers or students.

### Grade Overrides
- Filter and update any grade entry.

### Reports
- Generate summary reports.
- Export PDF/CSV/Excel or print.

### Notifications & Audit Logs
- View system notifications.
- Review audit logs for admin actions and grade changes.
